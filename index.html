    Import Fighters from CSV

    Bulk register fighters and auto-match them into divisions.

    REQUIRED CSV COLUMNS: Name, Club, Belt, DoB, Gender, Weight
    * DoB: Date of Birth (YYYY-MM-DD).
    * Weight: Numerical weight in kg (e.g. 73).
    * Belt: Rank (white, yellow, orange, green, blue, purple, brown, black).
    * Gender: Male or Female.
    * Optional columns: Type (Kumite/Kata). If missing, defaults to Kumite.
